# Programmatic Advertising Tools

Browser-based utilities for campaign setup, QA, and reporting workflows.

## Tools

- Campaign KPI calculator for CTR, CPC, CPA, and CPM
- UTM builder for GA4 campaign URLs
- Pixel QA checklist
- Programmatic campaign dashboard for pasted or uploaded mock data
- Ad tag formatter and validator
- OpenRTB-style JSON viewer

## Run locally

Open `index.html` in a browser. No build step or package install is required.
